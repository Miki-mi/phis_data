PhishNet utilizes Machine Learning technology in to immediately recognize phishing behaviours in each website you visit. Your browsing history or personal information will not be logged or sent to a third-party.

IP Address:

An IP address is used instead of the domain name in the URL.
Example: http://125.98.3.123/fake.html

URL Length:

Using a long URL to hide the doubtful part in the address bar.
Example: http://federmacedoadv.com.br/3f/aze/ab51e2e319e51502f416dbe46b773a5e/?cmd=_home&dispatch=11004d58f5b74f8dc1e7c2e8dd4105e811004d58f5b74f8dc1e7c2e8dd4105e8@phishing.website.html

Tiny URL:

Using URL Shortening Services to make URL considerably shorter in length.
Example: bit.ly/19DXSk4

@ Symbol:

Using “@” symbol in the URL leads the browser to ignore everything preceding the “@” symbol and the real address often follows the “@” symbol.

// Redirect:

The existence of “//” within the URL path means that the user will be redirected to another website.
Example: http://www.legitimate.com//http://www.phishing.com

- Prefix/Suffix:

The dash symbol is rarely used in legitimate URLs. Phishers tend to add prefixes or suffixes separated by (-) to the domain name so that users feel that they are dealing with a legitimate webpage.
Example: http://www.Confirme-paypal.com/

Sub Domains:

The number of sub domains the URL has.

HTTPS:

Checks if domain uses HTTPS (Hyper Text Transfer Protocol with Secure Sockets Layer)

Favicon:

A favicon is a graphic image (icon) associated with a specific webpage. If the favicon is loaded from a domain other than that shown in the address bar, then the webpage is likely to be considered a Phishing attempt.

Port:

It is much better to merely open ports that you need. If all ports are open, phishers can run almost any service they want and as a result, user information is threatened.

HTTPS in URL:

The phishers may add “HTTPS” to the URL in order to trick users. For
Example: http://https-www-paypal-it-home.soft-hair.com/.

Request URL:

Examines whether the external objects contained within a webpage such as images, videos and sounds are loaded from another domain.

SFH:

Server Form Handlers (SFH) that contain an empty string or is different from the domain name of the webpage are considered doubtful.

Mail:

Web form allows a user to submit their personal information that is directed to a server for processing. A phisher might redirect the user’s information to their personal email.

iFrames:

iFrame is an HTML tag used to display an additional webpage into one that is currently shown. Phishers can make use of the “iframe” tag and make it invisible.
